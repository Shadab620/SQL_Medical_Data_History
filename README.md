# SQL_Medical_Data_History

## OBJECTIVE
This project focuses on analyzing medical (healthcare) data to identify patterns
related to patients, treatments, and costs.
The objective is to transform raw medical data into meaningful insights 
that can support data-driven decision-making in the healthcare domain.

## Tool Used
Excel (Data Cleaning , Analysis)
PowerBI (Dashboard,Visualization)
SQL (Extracting ,Querying ,Reporting)


## Process
Cleaned raw data in excel
Handled missing values and format and duplicates
Used SQL queries to extract to clean and understand the data 
Built interactive dashboard in POWERBI
Presented insights using PowerPoint
